Para poder ejecutar el codigo python.

- DESCARGA E INSTALACION PARA SISTEMAS WINDOWS
	1) Se debe descargar de la pagina oficial de python la version 2.X y ejecutar el instalador siguiendo los pasos.
- PARA LINUX
	1) Si se posee una distro basada en Ubuntu 16.04 LTS no hara falta ya que viene integrado. En caso 
	contrario abrir la consola y escribir los siguientes comandos para instalar el paquete.
	
		wget --no-check-certificate https://www.python.org/ftp/python/2.7.11/Python-2.7.11.tgz
		tar -xzf Python-2.7.11.tgz  
		cd Python-2.7.11
	2) luego proceder a la configuracion con lo siguiente
		./configure  
		make  
		sudo make install  
-LISTO!!!


-EJECUTAR EL PROGRAMA

1) Ingresar a la consola e ingresar lo siguiente ya posicionado en la ubicacion del archivo .py

	python TP6.py <nro de computadoras a simular>

	Ejemplo:
		python TP6.py 10

GRACIAS!!!
